Novidades & Atualizações

FiberMap Pro v9.1

🚀 v9.1 — Análise, Clientes & Storage
  • 🔥 Heatmap de atenuação — botão "Heatmap" na bandeja pinta TODOS os cabos do mapa pela % do budget consumido. Verde <30%, amarelo 50-75%, vermelho ≥90%, vermelho-escuro excede. Legenda no canto. Cabos com pior atenuação ficam mais grossos. Auto-atualiza ao mexer no projeto
  • 👥 Painel global de Clientes — botão na bandeja abre lista com TODOS os clientes do projeto, busca por nome/endereço/ONU/plano, status ativo/inativo, click no cliente dá zoom no elemento pai e abre edição
  • 📊 Export CSV de Clientes — baixa planilha pronta pra Excel com todos os dados (UTF-8 com BOM)
  • 📥 Import CSV de Clientes — sobe planilha pra cadastrar em lote. Botão "📋 Modelo" baixa template com exemplo
  • 🔍 Busca turbinada — agora pesquisa também CLIENTES (nome, endereço, telefone, plano, ONU/serial). Resultados mostram subtitle com info adicional. Click em cliente abre direto pra edição
  • 💾 IndexedDB para o projeto — sem mais limite de 5MB do localStorage. Projeto principal agora vai pra IndexedDB com fallback automático em localStorage. Suporta projetos enormes
  • ⚡ Heatmap e alertas auto-atualizam ao salvar — sempre vê o estado real
📦 v9.0 — Marco grande
  • 📄 Relatórios em PDF — agora gera 3 tipos de relatório profissionais:
    • Cabo individual (botão "📄 PDF" no modal do cabo) — info técnica + mapeamento de fibras + OLT conectada por fibra + atenuação calculada
    • Circuito PON (rota completa OLT → cliente) — trajeto numerado dos hops, lista de clientes no circuito
    • Projeto inteiro (botão "📄 Gerar Relatório PDF" na seção Relatórios da bandeja) — 10 seções: Sumário Executivo, Distribuição de elementos, Status da Rede (ocupação OLT/MPLS/CTO, postes com elementos vinculados, conexões órfãs), Cabos por tipo (qtd + km), OLTs, MPLS, CTOs, Derivações, Emendas e Cabos detalhados com atenuação
  • 👤 Modelo de Clientes/ONUs — transforma o app em ferramenta operacional FTTH:
    • Cada porta de CTO ou Derivação pode ter um cliente vinculado (nome, endereço, telefone, plano, ONU/Serial, observações, status ativo/inativo)
    • Abrir CTO/Deriv mostra tabela completa de portas — porta livre é click pra criar, porta ocupada é click pra editar
    • Clientes aparecem automaticamente na árvore lateral dentro do circuito da PON (Armário → OLT → PON → Cabo → Postes → 👤 Cliente)
    • Modal de criação/edição com validação, geração de ID automático, status ativo/inativo
    • Relatório PDF de circuito mostra todos os clientes naquela rota com dados completos
📦 Novidades da v8.9 (Quick Wins)
  • 💾 Auto-backup em IndexedDB — a cada salvamento, snapshot datado é gravado. Mantém os 20 mais recentes. Abra "⏱ Backups" na bandeja ou aperte F3 para listar e restaurar
  • 📸 Snapshot rápido (F2) — exporta o projeto em JSON para download imediato com timestamp no nome
  • ● Indicador "Salvando…" no rodapé fica âmbar enquanto persiste, volta a verde sincronizado
  • ⌨️ Modal de Atalhos (?) — lista todos os atalhos do app
  • 📂 Drag & drop de arquivos — arraste KML/KMZ/JSON/CSV direto pro mapa
  • 🔒 Modo somente-leitura — adicione ?view=1 na URL para compartilhar visualização sem botões de edição
  • 📋 Coordenadas clicáveis — no modal de propriedades, click em lat/lng copia pro clipboard. Linha "Coords" copia lat,lng pronto pra colar no Google Maps
  • ⚠️ Confirmação reforçada para "Limpar" — precisa digitar LIMPAR + mostra contagem do que vai apagar
  • 🖨️ Modo Impressão — botão "Imprimir mapa" remove a interface e abre direto a vista limpa; Ctrl+P para imprimir/PDF
  • 👁 Toggles de camada — bandeja de ferramentas tem botões para mostrar/ocultar Postes, Armários e Cabos do mapa
  • ⚠️ Banner de alertas automático no topo direito: cabos com atenuação acima do budget, conexões órfãs detectadas. Click no banner abre lista completa, click num item dá zoom no elemento
